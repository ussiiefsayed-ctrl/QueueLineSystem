# QueueLineSystem
A C++ project that simulates a Queue Line Management System built on top of custom-designed data structures including a Doubly Linked List, Dynamic Array, Queue, and Stack implementations using templates.

The project also includes advanced utility classes such as String manipulation and Date handling, supporting operations like formatting, parsing, comparison, calendar generation, and time calculations.

It demonstrates strong OOP principles, generics (templates), memory management, and integration with STL containers where needed, making it a solid practice project for understanding data structures and system design in C++.
